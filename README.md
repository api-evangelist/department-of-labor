# Department of Labor (department-of-labor)
The U.S. Department of Labor (DOL) is the federal department that fosters, promotes, and develops the welfare of wage earners, job seekers, and retirees, improves working conditions, advances opportunities for profitable employment, and assures work-related benefits and rights. DOL exposes a portfolio of public APIs and data feeds including the modernized DOL APIv4 served from the DOL Open Data Portal, the Bureau of Labor Statistics Public Data API, the DOL Enforcement Data site, and Data.gov.

## APIs

### DOL Open Data API V4
Modernized REST API exposing 200+ DOL datasets, served from dataportal.dol.gov.

**Human URL:** [DOL Data Portal](https://dataportal.dol.gov/)

### BLS Public Data API V2
Programmatic access to BLS time series with up to 20 years of data per request.

**Human URL:** [BLS Developers](https://www.bls.gov/developers/)

### DOL Enforcement Data
WHD, OSHA, MSHA, OFCCP, and EBSA enforcement records.

**Human URL:** [DOL Enforcement Data](https://data.dol.gov/)

### DOL API Sampler
Interactive playground for exploring DOL APIv4 endpoints.

**Human URL:** [DOL API Sampler](https://devtools.dol.gov/apisampler)

### DOL Open Data Catalog
DOL datasets on Data.gov via CKAN.

**Human URL:** [Data.gov DOL](https://catalog.data.gov/organization/dol-gov)
